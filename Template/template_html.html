Псевдоэлементы в CSS

псевдоэлементы - элементы которых не существует в DOM дереве

  1. Текстовые псевдоэлементы - стилизация первой буквы текста ::first-letter 

                .my_class1::first-letter {
              font-size: 28px;
              color: blue;
            }
